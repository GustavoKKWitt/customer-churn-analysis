# 📊 customer-churn-analysis
This project analyzes customer data to understand why customers cancel their subscriptions (churn). 

---

# 🎯 Objective

Use data analysis to identify factors that increase the probability of customer cancellation.

---

# 🧠 Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# 📊 Analysis Performed

The project analyzes several variables related to customer behavior:

- Age
- Gender
- Call center interactions
- Payment delay
- Subscription duration
- Total amount spent
- Last customer interaction

---

# 🔎 Key Insights

### Age
Customers **above 50 years old showed a high cancellation rate**.

Possible solution:
- create services or support focused on older customers

---

### Gender
**Female customers canceled more than male customers.**

Possible solution:
- investigate product experience differences between groups
